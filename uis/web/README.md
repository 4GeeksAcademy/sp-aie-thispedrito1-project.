# UI Web - Incidents Analysis

Interfaz simple para:

- Subir archivo CSV
- Ejecutar analisis via `POST /api/incidents/analyze`
- Ver resumen de metricas en pantalla
- Descargar CSV via `GET /api/incidents/results/export`

## Uso rapido

1. Levanta la API en `services/api`.
2. Abre `uis/web/index.html` en el navegador.
3. Sube un CSV y pulsa `Analizar CSV`.
